# Latest_Feature_Practice

Overview

This folder contains practice code and examples for the latest Java language features used in this repository (modules, pattern matching, compact source files, records, text blocks, etc.).

Purpose

- Collect small examples and experimental snippets for recent Java versions.

How to use

1. Open the folder and pick an example file.
2. Compile with `javac` and run with `java` for the targeted Java version.

Notes

- Some examples require newer Java versions (17, 21, 22, 25). Use the appropriate JDK when compiling.
